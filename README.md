## 📖 Sobre o Projeto

O **Mapp7** é uma aplicação mobile desenvolvida como Trabalho de Conclusão de Curso, com o objetivo de conectar usuários que precisam de assistência mecânica a profissionais disponíveis para atendimento.

A proposta do projeto é oferecer uma plataforma simples, rápida e eficiente para solicitação de serviços mecânicos, permitindo que clientes encontrem suporte quando necessário e que mecânicos tenham uma nova oportunidade de trabalho e divulgação de seus serviços.

---

## ✨ Principais Funcionalidades

* 👤 Cadastro e autenticação de usuários
* 🔧 Solicitação de serviços mecânicos
* 📍 Localização de profissionais
* 📱 Interface mobile intuitiva
* 🚗 Gerenciamento de atendimentos
* 🔐 Controle de acesso e dados do usuário

## 🚀 Como Executar

### 1. Instalar as dependências

```bash
npm install
```

### 2. Iniciar a aplicação

```bash
ionic serve
```

### 3. Acessar no navegador

```text
http://localhost:8100
```

### 4. Executar em dispositivo ou emulador

```bash
ionic cordova run android
```

ou

```bash
ionic cordova run ios
```

ou

```bash
ionic cordova run browser
```
